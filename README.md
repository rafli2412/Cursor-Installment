# Cursor Installment

Documentation of the Cursor IDE setup, extensions installed, and steps completed during initial configuration.

## Tools Installed

| # | Tool / Extension | Platform | Usage |
|---|------------------|----------|-------|
| 1 | **Cursor IDE v3.6.31** | Windows | AI-powered code editor based on VS Code. Use it as your primary IDE for writing, editing, and navigating code. Open a folder or clone a repo, then use the built-in chat and agent features for coding assistance. |
| 2 | **Claude Code Extension** | Cursor IDE | Adds Anthropic's Claude as an AI coding assistant inside Cursor. Use it from the extension panel or command palette to get code suggestions, explanations, and refactors powered by Claude. |
| 3 | **Codex Extension** | Cursor IDE | Integrates OpenAI Codex for AI-assisted coding. Use it to generate, complete, and edit code directly within the editor via prompts and inline suggestions. |
| 4 | **Prettier Extension** | Cursor IDE | Code formatter for consistent style across JavaScript, TypeScript, JSON, CSS, and more. Enable **Format on Save** in settings, or run **Format Document** (`Shift+Alt+F`) to apply formatting. |

## Steps Completed

1. **Installed Cursor** — Downloaded and installed Cursor IDE v3.6.31 on Windows.
2. **Added Claude on Cursor** — Installed and configured the Claude Code extension.
3. **Added Codex on Cursor** — Installed and configured the Codex extension.

## Issues Found

| # | Issue | Resolution |
|---|-------|------------|
| 1 | Authentication problem when integrating Claude and Codex | Resolved by clearing the extension cache and updating previously installed extensions from VS Code. |

## Quick Start

1. Open **Cursor IDE** and sign in if prompted.
2. Open the **Extensions** view (`Ctrl+Shift+X`) to verify Claude, Codex, and Prettier are installed and enabled.
3. For Claude or Codex, complete authentication when prompted (sign in with your provider account).
4. Open a project folder and start coding — use Cursor's chat, Claude, or Codex for AI assistance as needed.
5. Let Prettier format your files on save or on demand to keep code style consistent.
